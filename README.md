# Trivia Game

## Built With;

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Jquery

## ![Screenshot of game](assets/images/screenshot.jpg)

## Game rules:

- One question at a time - Multiple choice answers.
- 15 second countdown timer for each question.
- Results displayed at end of round.
